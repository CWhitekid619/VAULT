# VAULT

Personal movie & TV show tracker. Built as a single HTML file, no backend, no subscription.

- Search films and series via TMDB
- Log what you watched into 5 buckets (Richtig gut → Schlecht)
- Head-to-head Elo ranking system
- Custom lists, random picker, series progress tracking
- Runs as PWA on iPhone (Add to Home Screen)

Built with vanilla HTML/CSS/JS. Data stored in localStorage.
